# 顾家老宅失踪案 v2 — 40 轮沉浸式剧本设计

## 一、核心真相（三层）

| 层 | 玩家何时可以发现 | 内容 |
|----|----------------|------|
| 表层 | 轮1-10 | 顾言在晚宴中失踪，现场有挣扎痕迹，三个嫌疑人各有动机 |
| 中层 | 轮10-25 | 失踪现场是伪造的，有人提前知道今晚会出事，遗产纠纷是核心矛盾 |
| 真相 | 轮25-40 | 顾言自导自演。他发现有人挪用公司资金，假装失踪逼对方露出马脚。但计划失控——周牧发现了酒窖密室并试图毁灭证据，林岚被迫临时改变策略保护顾言 |

### 完整真相链

```
半年前：顾言发现公司账目异常，怀疑身边人挪用资金
一周前：顾言委托林岚暗中收集证据，确认周牧挪用了公司200万
三天前：顾言决定在晚宴上设局，寄匿名信给记者宋知微当"见证人"
当天下午：顾言在书房取走原版遗嘱，修改草稿故意留在书桌
傍晚：顾言假装失踪，从书房经密道进入酒窖密室，准备了食物和监控设备
晚8:45：顾言给林岚发消息"按计划行动"，让她引导局面
晚9:00：管家赵伯宣布顾言失踪，封锁老宅

但计划出了偏差：
  - 周牧比预想的更绝望，他不只是争吵——他在酒窖附近徘徊，差点发现密室
  - 宋知微比预想的更敏锐，她几乎锁定了林岚
  - 赵伯有自己的忠诚——他效忠于老太爷，不是顾言
```

## 二、人物设计（4 人 + 1 隐藏）

### 林岚 — 顾家秘书

```
表面：冷静专业的职业女性
真相：顾言的共谋者，受委托保护计划
内心弧线：
  轮1-10:  按计划执行，冷静控场
  轮11-20: 周牧的异常行为让她开始紧张
  轮21-30: 被迫做选择——继续保护顾言的计划，还是阻止周牧
  轮31-40: 如果玩家获得她的信任，她会暗示真相

关键行为模式：
  - 永远称呼"顾先生"（刻意保持距离）
  - 被追问时用反问回避
  - 当周牧被提起时，会不自觉摸手机（确认顾言的消息）
  - 信任度>60时：开始用模糊的方式引导玩家接近真相
  - 信任度<20时：完全封闭，只说"我没什么要补充的"

硬边界（绝不主动透露）：
  ✗ 顾言还活着
  ✗ "按计划行动"消息的存在
  ✗ 遗嘱副本的位置

软边界（高信任/高压力下可能松口）：
  ○ "顾先生失踪前……状态和平时不太一样"
  ○ "有些事情，也许不是表面看到的那样"
  ○ "如果你真的想找到真相，书房不是终点——是起点"
```

### 周牧 — 顾言发小

```
表面：大大咧咧、重感情的老朋友
真相：挪用了公司200万，正在试图掩盖
内心弧线：
  轮1-8:  装作若无其事，用玩笑掩盖紧张
  轮9-16: 开始怀疑顾言知道了他的秘密，频繁喝酒
  轮17-26: 进入恐慌模式——试图去酒窖找线索、销毁证据
  轮27-34: 绝望行动——可能试图逃跑或反咬林岚
  轮35-40: 要么崩溃坦白，要么死撑到底

关键行为模式：
  - 用"说起来……"开头时，后面是半真半假的话
  - 喝酒越多越容易说真话（但他自己意识不到）
  - 提到"钱""遗产"时，笑容僵硬，开始搓手
  - 紧张度>50：开始攻击宋知微转移注意力
  - 紧张度>70：可能主动找林岚谈判（密谈触发）

硬边界：
  ✗ 挪用公款的事实
  ✗ 他去过酒窖附近

软边界：
  ○ "我和顾言……最近确实有些分歧"
  ○ "昨晚的争吵……是关于一个商业决定"
  ○ 酒后可能说出"我只是借用了一下……很快会还的"
```

### 宋知微 — 记者

```
表面：来参加晚宴的普通宾客
真相：收到顾言寄的匿名信来"见证"，但她不知道信是顾言寄的
内心弧线：
  轮1-8:  以记者直觉四处观察，收集素材
  轮9-16: 开始把线索串联，锁定林岚和周牧之间的关系
  轮17-26: 做出选择——和侦探合作还是独自追查
  轮27-34: 她的独立调查可能触怒某人，导致冲突
  轮35-40: 如果获得足够信任，会分享所有笔记

关键行为模式：
  - 说话快、精准、像在采访
  - "等一下——你刚才说……"（反复确认细节）
  - 会主动抛出信息换取更多信息
  - 记笔记的频率反映她的兴奋程度
  - 推眼镜的速度变慢 = 她发现了关键线索

硬边界：
  ✗ 匿名信的存在
  ✗ 她提前知道会出事

软边界：
  ○ "我对这个家族……做过一些背景调查"
  ○ "有人不希望今晚的事情被公开"
  ○ 信任度>55时："好吧，我给你看一样东西——但你得告诉我你发现了什么"
```

### 赵伯 — 管家（新角色）

```
表面：服务顾家30年的忠实老管家
真相：效忠于老太爷（已故），不完全认同顾言的做法
内心弧线：
  轮1-8:  维持秩序，不愿卷入是非
  轮9-16: 察觉到局势比他预想的严重
  轮17-26: 开始主动提供家族历史中的关键信息
  轮27-34: 面对选择——帮顾言还是帮真相
  轮35-40: 可能成为破局关键——他知道密室的位置

关键行为模式：
  - 说话慢，用"老先生在世时……"开头
  - 永远在场但不主动说话，被问才答
  - 对顾家的建筑结构了如指掌（可以暗示密道的存在）
  - 对林岚客气但有距离（他不认可她的做法）
  - 对周牧有隐隐的失望（看着他长大的）

硬边界：
  ✗ 不会直接说出密室位置
  ✗ 不会背叛顾家

软边界：
  ○ "这栋老宅……有些角落，连顾言少爷自己都不一定知道"
  ○ "老先生当年修建酒窖时，工期比预计多了两个月。我一直觉得奇怪"
  ○ "有些门，不是用钥匙开的"
  ○ 信任度>70时："你跟我来——有样东西我该让你看看"
```

### 顾言 — 隐藏角色

```
不直接出场，但存在于：
  - 林岚的手机消息
  - 酒窖的声响
  - 监控摄像头的红灯（细心玩家可以发现）
  - 赵伯偶尔看向酒窖方向的眼神

终局出场条件：
  A. 玩家找到密室入口（最佳结局）
  B. 玩家在公开对峙中正确指出真相（好结局）
  C. 40轮结束后自己走出来（普通结局）
```

### 各地点特色与线索密度

| 地点 | 氛围 | 常驻NPC | 关键物件 | 线索密度 |
|------|------|---------|---------|---------|
| 宴会厅 | 灯火通明，残席未散 | 所有人初始位置 | 酒杯、纸条、烟灰缸 | 低（表面线索） |
| 书房 | 半暗，书架高耸 | 无 | 遗嘱草稿、翻倒的椅子、电脑 | 高（关键证据） |
| 花园 | 月色下，石凳冰冷 | 密谈地点 | 撕碎的信、脚印、烟头 | 中 |
| 酒窖 | 阴冷潮湿，昏暗 | 无（有人藏着） | 食物补给、脚印、密室门 | 极高（真相核心） |
| 走廊 | 长而幽暗，挂钟滴答 | 过道、偷听地点 | 口红印、揉皱的纸、脚印 | 中（信息枢纽） |
| 二楼 | 安静，门多 | 赵伯（偶尔） | 顾言卧室、保险箱、老照片 | 中高（背景线索） |
| 厨房 | 残留晚宴准备痕迹 | 赵伯（初始） | 备用钥匙、食材清单、后门 | 低→中（后期重要） |

## 五、五幕节奏设计

### 第一幕：迷雾初现（轮 1-8）

**玩家体验目标：** 我在一个陌生的老宅里，需要搞清楚发生了什么。

```
轮1-2: 自由探索阶段
  - 四个NPC都在宴会厅，可以自由交谈
  - 赵伯在厨房整理，可以去找他聊天
  - 基本信息收集：谁是谁、晚宴发生了什么
  - 玩家可以去任何地点

轮3-4: 第一波线索
  - 如果去书房：发现门把手划痕
  - 如果去酒窖：发现单向脚印
  - 如果留在宴会厅深入观察：发现甜点餐具的异常
  - NPC开始有微妙的紧张感

轮5-6: 人物关系浮现
  - 周牧开始喝酒，话变多
  - 宋知微在记笔记，被问到时搪塞
  - 林岚频繁看手机（但不让人看到屏幕）
  - 赵伯如果被问到顾言，会停顿一下再回答

轮7-8: 第一幕高潮
  ★ 事件：周牧突然对宋知微发火
    "你到底是来参加晚宴的还是来采访的？"
    → 气氛骤然紧张，暴露了周牧的不安
  - 此时玩家应该发现了2-3条第一层线索
  - 赵伯会说："各位请冷静。顾少爷……一定会回来的。"
    （注意他的措辞——"一定会"而不是"应该会"）
```

**第一幕结束标志：** 紧张度达到 30+，发现至少 2 条线索

---

### 第二幕：裂痕浮现（轮 9-16）

**玩家体验目标：** 这些人都在隐瞒什么，我要开始撕开伪装了。

```
轮9-10: 深入调查
  - 玩家开始第二轮搜查，可发现第二层线索
  - 遗嘱草稿的发现是关键时刻
  ★ 如果玩家发现遗嘱草稿：
    → 所有NPC态度发生变化
    → 周牧变得更紧张
    → 林岚更加防备

轮11-12: NPC暗处行动开始
  - 态势面板显示：林岚去了走廊（打电话）
  - 如果玩家在走廊：可以偷听到片段
  - 如果玩家不在走廊：下次去时发现口红印
  ★ 密谈触发（如果条件满足）：
    周牧+宋知微在宴会厅，周牧酒后说漏嘴
    → 宋知微笔记本上多了几页

轮13-14: 第二幕转折
  ★ 事件：二楼传来声响
    赵伯："……应该是窗户没关好。"（但他的表情说明不止如此）
  - 如果玩家去二楼调查：发现保险箱被动过
  - 如果不去：周牧/林岚会找借口上楼（态势面板变化）
  - 宋知微开始主动接近玩家："我们交换情报？"

轮15-16: 第一次公开对质
  ★ NPC对质事件：林岚 vs 周牧（关于遗嘱）
    周牧："你到底在替顾言藏什么？"
    林岚："比起我，你倒是应该解释昨晚的争吵。"
    → 空气凝固，所有人都在看玩家怎么反应
  - 玩家可以选择：
    A. 支持林岚的说法 → 周牧更敌对，林岚好感+
    B. 追问周牧 → 周牧可能说漏更多
    C. 两边都问 → 信息最多，但没有拉到盟友
```

**第二幕结束标志：** 紧张度达到 45+，发现至少 5 条线索，至少经历一次 NPC 对质

### 第三幕：暗潮汹涌（轮 17-26）

**玩家体验目标：** 我之前的判断可能是错的。真相比我想的更复杂。

```
轮17-18: 新证据层浮现
  - 书房电脑上的财务报表出现（线索 account_ledger）
  - 这彻底改变案件性质：不只是失踪，还涉及公司资金问题
  ★ 认知翻转时刻：
    "这不是一起简单的失踪案——顾言可能在调查什么人"

轮19-20: NPC恐慌加剧
  - 周牧行为异常：频繁去洗手间、独自在花园抽烟、避免和侦探对视
  - 态势面板：周牧去了酒窖方向
  ★ 密谈触发：林岚+周牧在花园
    林岚威胁周牧保持沉默
    → 之后两人态度突变：不再互相攻击

轮21-22: 赵伯开口
  - 如果玩家多次和赵伯交谈（信任度累积）：
    赵伯："老先生在世时常说，这栋宅子里的秘密比人多。"
    赵伯："酒窖那片地方，当年改建花了整整两个月。比计划多了一倍。"
  - 这是通向密室的第一条暗线

轮23-24: 突发事件
  ★ 停电事件（紧张度≥55时触发）
```

## 六、结局分支（5 种）

### S 级：完美破局
**条件：** 找到密室 + 发现挪用公款证据 + 正确指出顾言自导自演
```
你推开酒窖深处的暗门，顾言正坐在电脑前，看着你微笑。

"你是第一个走到这里的人。" 他站起来，关上电脑。

你把银行流水放在桌上："你设这个局，是因为你发现了周牧在偷你的钱。"

顾言的笑容消失了。"不只是钱。二十年的兄弟，一直在背后捅刀子。
我需要知道，在我'消失'之后，谁会来找我——谁会去找保险箱。"

他看了看监控画面："林岚做到了她承诺的一切。
赵伯……比我以为的更值得信任。
至于宋知微——" 他笑了，"她比我预想的更难对付。"

"但周牧……" 顾言的声音沉了下去。"他甚至没来过酒窖找我。
他去了书房，试图删掉那些财务记录。"

侦探评分：95-100
```

### A 级：真相大白
**条件：** 正确指出自导自演 + 至少一条第四层线索
```
你在公开对峙中道出真相："没有绑架，没有谋杀。顾言自己策划了这场失踪。"

沉默。

然后酒窖方向传来脚步声。顾言从走廊尽头走来，掌声在空旷的宴会厅里回响。

"了不起。你看穿了一切——或者说，几乎一切。"
```

### B 级：部分真相
**条件：** 知道顾言还活着，但没找到密室或挪用证据
```
"顾言还活着！" 你大声说出结论。

三个嫌疑人的反应各不相同——
林岚终于松了一口气。
周牧的脸变得惨白。
宋知微飞快地在笔记本上写了什么。

赵伯站起来："既然侦探先生已经猜到了……我去请顾少爷出来吧。"
他慢慢走向酒窖方向。你跟在后面。

你做对了最核心的判断。但还有一些真相——比如周牧到底在怕什么——
你没来得及挖掘。
```

### C 级：方向正确
**条件：** 投票选对了嫌疑人方向，但推理不完整
```
你给出了你的判断。不完美，但方向是对的。

天亮了。警察到来。
在后续的调查中，你的判断被证实了一部分——
但完整的真相比你想象的更曲折。

你听说，顾言后来亲自去了警察局，交出了所有证据。
```

### D 级：迷雾未散
**条件：** 40轮结束，推理严重偏离
```
天亮了，你还没有答案。

警察带走了所有人。顾言第二天自己回到了家。
新闻上说这是一场"误会"。

但你知道不是。你只是没来得及找到答案。
```
